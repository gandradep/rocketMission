# React and Redux Space Travelers' Hub

> This is a spaceship demo web app. Users can reserve a rocket an join space missions.

## Built With

- React and Redux

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Have a Desktop or Laptop Computer running on with Windows, Mac OS or Linux operating system.
- Have git installed on the Computer
- Have a browser installed on the Computer

### Install

- To clone the project on your machine: Type `git clone https://github.com/gandradep/rocketMission.git`.

- To build project files in development mode _(the destination folder is /dist)_:
  `npm run build`
- To run the project on webpack live server:
  `npm run start`
